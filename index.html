What is JavaScript?

JavaScript (JS) is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions. While it is most well-known as the scripting language for Web pages, many non-browser environments also use it, such as Node.js, Apache CouchDB and Adobe Acrobat. JavaScript is a prototype-based, multi-paradigm, single-threaded, dynamic language, supporting object-oriented, imperative, and declarative (e.g. functional programming) styles. Hello World!

This is the simplest JavaScript code

console.log(“Hello World”);

The console.log(arg) command prints out the argument in the web browser’s console. This command is useful in various situations for testing purposes.

Variables

What are variables?

Variables are place holders. They hold values under specific identifiers. For example:

x = 5;
Variable_x = 6;

In the first case, the identifier is “x” and the value that this identifier holds is 5. Similarly, in the second case, the identifier is “Variable_x” and the value that this identifier holds is 6.

Declaring variables

Now that we know what variables are, before we can use them, we need to declare the variable by giving it an identifier and keyword. You can declare a variable in one of the three following ways:

var Name; //used for both global and local scope
let Name; //used in only local scope
const Name; //identify a constant variable, a variable that cannot change throughout the code

To assign values to a variable, use the assignment operator “=”. For example:

Name = “Emma”;

Or

var Name = “Emma”;
Variable scope

The scope of a variable is it’s availability with respect to the rest of the code. A global (scope) variable can be seen and accessed anywhere in the code, meanwhile a local (scope) variable can only be seen and accessed within its’ function (see function section).
For example:

if(true){ var x = 5; console.log(x); // console will show: 5 }
console.log(x); //console will show:5
if(true){ let y = 5; console.log(y); //console will show: 5 }
console.log(y) // console will show : Reference Error : y is not defined

If...else if...else Statements

In simple words, condition statements are a set of instructions that tells you code how it should make decisions. For example, “If I have 5 dollars or more, then I will buy apples. If I have less than 5 dollars, then I will buy cherries. If I have no money, then I won’t buy anything.”
This sentence can be translated to the following condition statement:

if( money >= 5){ console.log(“Buy Apples”); }
else if (money < 5){ console.log(“Buy cherries”); }
else{ //if none of the above conditions are satisfied
console.log(“Not buying anything”); }

In general, conditions are written in the following format. Note that you can have as many else if statements as you see fit, that means you can also have no else if statement.

If (condition){ execute code; }
else if(condition){ execute code; }
else{ execute code; }

Loops

Loops are a block of code that will be executed repeatedly when certain conditions are met. There are many different types of loops that are available for use, this documentation will specifically look at the while loop. In general, the while loop has the following struc

while(condition){
execute code;
}
//Sample code:
while (true){
console.log(“true”);
//when the condition is true, print “true” in the console.
}

Functions

A function definition (also called a function declaration, or function statement) consists of the function keyword, followed by:

  • The name of the function.
  • A list of arguments to the function, enclosed in parentheses and separated by commas.
  • The JavaScript statements that define the function, enclosed in curly brackets, { }.

For example, the following code defines a simple function named square:

function square(number) { return number * number; }

The function square takes one argument, called number. The function consists of one statement that says to return the argument of the function (that is, number) multiplied by itself. The return statement specifies the value returned by the function.

return number * number;

Primitive parameters (such as a number) are passed to functions by value; the value is passed to the function, but if the function changes the value of the parameter, this change is not reflected globally or in the calling function.
